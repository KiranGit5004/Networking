# Network Troubleshooting

## what is troubleshooting
troubleshooting is the process of finding and fixing network issues systematically not randomly.

### Flow
IP → Gateway → Internet → DNS → Service

### Steps
1. Check IP (ip addr)
2. Check gateway (ip route)
3. Ping gateway
4. Ping internet (8.8.8.8)
5. Check DNS (nslookup)
6. Check service/port

### Commands
- ip addr
- ip route
- ping
- nslookup
- dig
- ss -tuln
- traceroute
- telnet
- curl
- ssh
- scp - secure copy files between machines
- ps - list your process
- ps aux - list all processes
