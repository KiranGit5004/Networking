# Subnetting Interview Questions

1. What is subnetting?
Answer: Dividing a network into smaller subnets.

2. What is CIDR?
Answer: Method of representing IP using / notation.

3. How many IPs in /24?
Answer: 256 total, 254 usable.

4. What is network and broadcast?
Answer: First and last IP of subnet.

5. What is block size?
Answer: 256 - subnet mask.

6. How many subnets in /26?
Answer: 4

7. Usable IPs in /27?
Answer: 30

8. How many hosts in /26?
2^(32-26) - 2 = 62

4. What is broadcast address?
Last IP of a subnet used to communicate with all devices.

5. Difference between CIDR and Subnet Mask?
CIDR → /24  
Subnet Mask → 255.255.255.0
