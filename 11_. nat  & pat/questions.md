# INTERVIEW QUESTIONS + ANSWERS

❓ What is NAT?

✅ Answer:
NAT translates private IP addresses into public IP addresses for internet communication.

❓ What is PAT?

✅ Answer:
PAT allows multiple devices to share a single public IP using different port numbers.

❓ Difference between NAT and PAT?

✅ Answer:
NAT maps one-to-one, while PAT maps many-to-one using ports.

❓ Why NAT is used?

✅ Answer:
To conserve public IPs and allow private networks to access the internet.

❓ What is port forwarding?

✅ Answer:
It allows external access to an internal device through NAT.

❓ What is double NAT?

✅ Answer:
When two routers perform NAT, causing connectivity issues.
