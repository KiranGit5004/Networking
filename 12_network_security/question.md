# INTERVIEW QUESTIONS + ANSWERS

❓ What is firewall?

✅ Answer:
A firewall is a security system that controls incoming and outgoing network traffic based on rules.

❓ Why firewall is used?

✅ Answer:
To protect systems from unauthorized access and attacks.

❓ What is port?

✅ Answer:
A logical endpoint used by services for communication.

❓ What happens if port is closed?

✅ Answer:
Service becomes inaccessible.

❓ What is least privilege?

✅ Answer:
Giving only necessary access to reduce risk.

❓ What is port scanning?

✅ Answer:
Technique used to find open ports in a system.
