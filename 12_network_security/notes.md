# Network Security Basics

## Network Security?
network security means protecting network and systems from:
unauthorized access, attackers, data breaches.

## Firewall
A firewall filters incoming and outgoing traffic between networks.
firewall permits or denies traffic based on predefined security rules and policies.

## Types
- Packet filtering
- Stateful
- Application

## Ports
A port is logical endpoints used by services for communication.

- 22 → SSH
- 80 → HTTP
- 443 → HTTPS

## Actions

| Action | Meaning        |
| ------ | -------------- |
| Allow  | Permit traffic |
| Deny   | Block traffic  |
| Reject | Block + notify |

## VPN
a Virtual Private Network (VPN) creates a secured encrypted tunnel between user and private network
used for: secure remote access and private cloud acces.
