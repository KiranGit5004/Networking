# Routing Interview Questions

1. What is routing?
Answer: Routing is the process of forwarding data packets from one network to another using routers.

2. What is router?
Answer: A router is a Layer 3 device that connects different networks and forwards packets.

3. What is default gateway?
Answer: Default gateway is the IP address of a router used to send traffic outside the local network.

4. What is routing table?
Answer: A routing table contains rules that determine where packets should be forwarded.

5. What is default route?
Answer: Default route (0.0.0.0/0) is used when no specific route matches the destination.

6. Static vs dynamic routing?
Answer: Static routing is manually configured, while dynamic routing automatically updates routes using protocols.

7. What happens if gateway is wrong?
Answer: System cannot access external networks (no internet).
