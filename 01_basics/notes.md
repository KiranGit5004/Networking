# Networking Basics

## What is Network
A computer network is a group of connected devices used to share data and resources.

## Types of Network
- LAN: Small area (home, office)
- WAN: Large area (internet)
- MAN: City level

## Topologies
- Star (most common)
    All devices connected to one central device (switch/router)
- Bus (old)
    Single cable backbone
- Mesh (reliable)
    Devices connected to each other
- Ring (rare)
    Circular connection

## Key Terms
- Bandwidth: Max capacity
- Latency: Delay
- Throughput: Actual speed
👉 Example:
You have 100 Mbps plan but get 40 Mbps → that’s throughput

## MAC vs IP
- MAC: Physical address
- IP: Logical address

## Communication Types
- Unicast: One to one
- Broadcast: One to all
- Multicast: One to group

## Question
Why do we need IP if MAC already exists?
👉 Because MAC works locally, IP works globally
Which network type is the Internet?
👉 WAN
