# Protocol Interview Questions

## Basic
1. What is DNS?
DNS translates domain names into IP addresses.

2. What is DHCP?
DHCP automatically assigns IP addresses using DORA process.

## Medium
3. HTTP vs HTTPS?
HTTP is unencrypted, HTTPS uses SSL/TLS for secure communication.

4. What is SSH?
Secure remote login.

## Advanced
5. What happens when you open website?
DNS resolves domain → TCP connection → HTTP request → server response.

6. What is port?
A port is a logical endpoint used to identify services.
