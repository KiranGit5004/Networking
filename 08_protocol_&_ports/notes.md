# Network Protocols

## Protocol
A protocol is a set of rules that defines how devices communicates over a network. 

## Port
A port is virtual point where network connections start and end
ports are software based and managed by a operating system
each port is associated with a specific service 

## HTTP/HTTPS
transfer web data between client and server.
- HTTP → 80
- HTTPS → 443

## DNS
- Converts domain to IP
- Port 53

## DHCP
- Assigns IP
- DORA process

## SSH
- Secure remote login   
- Port 22

## FTP/SFTP
- File transfer
- FTP → 21
- SFTP → 22

Difference:
FTP → insecure
SFTP → secure (SSH-based)
