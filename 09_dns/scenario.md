### Case 1:

Ping IP works but domain fails

👉 Example:

ping 8.8.8.8  ✅
ping google.com ❌

👉 Problem:
DNS not working

### Case 2:

Website opens slowly

👉 Possible:

Slow DNS server
High latency

### Case 3:

Wrong website opens

👉 Problem:

DNS cache poisoning
Wrong DNS entry

### Case 4:

Domain not resolving after change

👉 Problem:
DNS propagation delay
