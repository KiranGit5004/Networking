## INTERVIEW QUESTIONS + ANSWERS
1. What is DNS?

✅ Answer:
DNS translates domain names into IP addresses.

2. What is DNS hierarchy?

✅ Answer:
DNS uses root, TLD, and authoritative servers to resolve queries.

3. What is A record?

✅ Answer:
Maps domain to IPv4 address.

4. What is CNAME?

✅ Answer:
Alias of another domain.

5. What is MX record?

✅ Answer:
Used for email routing.

6. What is DNS cache?

✅ Answer:
Stored DNS results for faster access.

7. What is TTL?

✅ Answer:
Time a DNS record is cached.

8. What is DNS propagation?

✅ Answer:
Time taken for DNS changes to update globally.
