# Switching & VLAN

## Switching
- Works on Layer 2
- Uses MAC address
- Maintains MAC table

## Process of Switching
- Frame arrives
- Switch checks MAC table
- If known → forward to correct port
- If unknown → flood to all ports

👉 This flooding behavior is important

## VLAN
- Logical network segmentation

## Types
- Access Port → single VLAN
- Trunk Port → multiple VLANs

## Benefits
- Security
- Reduced broadcast
