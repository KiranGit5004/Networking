# DHCP Interview questions

❓ What is DHCP?

✅ Answer:
DHCP is a protocol that automatically assigns IP configuration to devices.

❓ What is DORA process?

✅ Answer:
Discover, Offer, Request, Acknowledge process used for IP assignment.

❓ What ports does DHCP use?

✅ Answer:
UDP ports 67 (server) and 68 (client)

❓ What is DHCP lease?

✅ Answer:
Time period for which IP is assigned.

❓ What is APIPA?

✅ Answer:
169.254.x.x address assigned when DHCP fails.

❓ What is DHCP relay?

✅ Answer:
Allows DHCP communication across networks.

❓ What is IP conflict?

✅ Answer:
Two devices having same IP.
